📜이용 방법


할 일 추가는 입력 칸에 할 일을 입력합니다.

수정 및 삭제는 할 일칸에 마우스를 올려 아이콘을 클릭합니다.

할 일 완료는 해당 칸을 클릭하여 완료 표시를 합니다.

전체 삭제는 입력 칸 위 버튼을 클릭합니다.

미완료된 할 일과 완료된 할 일이 우측 상단에 표기됩니다.

HOW TO MAKE


index.html

  • index.html에 iframe 태그를 넣어 화면 전환
  • 우측 스크롤 바 안보이게 설정
  • 우측 하단에 다크 모드 구현
  • 좌측 상단 사이드 바에 transform을 활용해서 애니메이션 구현

landingPage.html

  • 스크롤 시 opacity를 바꾸어 자연스럽게 연출
  • 우측에 버튼을 클릭하여 해당 위치로 스크롤
  • 이용 방법은 캐러셀을 만들어서 이미지로 설명함

toDoList.html

  • localStorage를 활용해서 페이지를 나가도 할 일이 로컬에 저장됨
  • 할 일 추가 시 내용, id, 완료 여부를 localStorage에 저장
  • 완료하면 할 일에 취소선이 생기고 회색으로 변경
  • 할 일 hover 시 수정, 삭제 아이콘 띄우기
  • 전체 삭제 시 localStorage 비우기
